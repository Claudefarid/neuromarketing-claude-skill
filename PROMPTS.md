# Ready-to-Use Prompts

Copy, fill the [brackets], paste into Claude.

**Audit copy:**
> Audit this copy with the neuromarketing skill. Buyer: [who]. Traffic: [cold/warm/hot]. Copy: [paste]

**Write an ad:**
> Write 3 Facebook ad variations for [product]. Target: [audience]. Their biggest fear: [fear]. Use different lead triggers per variation.

**Fix a landing page:**
> My landing page gets visitors but no sales: [URL or paste]. Diagnose what's killing conversion and rewrite the above-the-fold section.

**Design pricing:**
> Design a 3-tier pricing page for [product]. Current price: [X]. Use anchoring and a decoy tier. Explain the psychology.

**Build an offer:**
> Create a launch offer for [product] using real scarcity (here's what's genuinely limited: [X]) and a bonus instead of a discount.

**Rescue a boring post:**
> This post got no engagement: [paste]. Rewrite the hook 5 ways using different triggers, then rewrite the full post with the strongest one.
